Introduction.

Paxml is an XML-based scripting language, providing simple yet powerful syntax to automate your business operations. For instance, accessing database and files, driving browsers, calling web APIs and glue all these tasks together to a integral piece of logic to automate your manual operational work. Paxml is suitable for operational staff to automate business processes and for software testers to automate test executions.

Paxml is also an open source software released under Affero General Public Version 3.0 license.

Taste a bit

Here is a simple example to let Google translate website translate all rows (possibly 1000 rows) of column A of an Excel file from English to German, and write the German translation back to column B:

<scenario>

  <!-- open an excel file called "MyExcel.xls" and name it as excel -->
  <readExcel id="excel" file="MyExcel.xls" range="A1:"/>

  <!-- iterate through all rows of the excel, 
    and for each row, repeat a sequence of actions -->
  <iterate values="${excel}">
    <!-- open browser to access google transalte page -->
    <url>https://translate.google.com/#en/de</url>

    <!-- type the text of current row's A column 
      into the google translate page -->
    <type locator="source">${var.A}</type>    

    <!-- click on the "translate" button -->
    <click locator="gt-lang-submit"/>            

    <!-- pick the translated German text from the browser and write back to 
        the B column of the corresponding row of MyExcel.xls -->
    <writeExcel file="MyExcel.xls" row="${index+1}" column="B" >    
      <pick locator="result_box" timeout="5000"/>
    </writeExcel>
  </iterate>

  <!-- Voila, we are done!! -->

</scenario>
